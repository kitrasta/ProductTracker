# ProductTracker

Desktop-приложение для контроля сроков годности продуктов.

## Как работаем

Ты пишешь код, я помогаю как reviewer:
- проверяю архитектуру и UI-решения;
- даю короткий список правок по приоритету;
- предлагаю следующий безопасный шаг по реализации.

## Команды

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Мини-чеклист перед PR

- есть `loading / empty / error` состояния;
- есть валидация форм и понятные тексты ошибок;
- навигация работает с клавиатуры;
- даты и статусы отображаются единообразно.
